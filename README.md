# Mask Wearer Re-Identification Dataset

본 데이터 셋은 정보통신기획평가원 주관 '감시카메라 기반 마스크 착용자 얼굴인식 및 재인식 (Re-ID) 기술 개발' 과제의 결과물로, 마스크 착용자 대상의 사람 재인식 기술 개발을 위한 학습 및 테스트 용 대규모 이미지 및 ID(약 2천개) 정보를 제공합니다.
